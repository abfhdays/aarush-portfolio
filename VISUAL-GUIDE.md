## Color Palette

```
Background:    #fafafa  (light gray)
Text:          #171717  (near black)
Secondary:     #525252  (medium gray)
Border:        #e5e5e5  (light border)
Accent:        #171717  (same as text)
```

## Typography Hierarchy

```
h1:  2rem (32px) - Bold - Your name
h2:  1rem (16px) - Medium - Section titles (lowercase)
p:   Default     - Regular - Body text
```

## Interactive Elements

- Links: underlined with light gray, turns dark on hover
- Projects: title is clickable if link provided
- All transitions: 0.2s for smooth feel

## Spacing

- Max width: 2xl (672px) - readable line length
- Padding: 6 (24px) on mobile, maintained on desktop
- Section gaps: 12 (48px)
- Item gaps: 6 (24px)

Clean. Simple. Focused on content.
